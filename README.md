# 💪 Telegram Бот «Отжимания Challenge»

Бот для группового контроля ежедневных отжиманий.  
**Каждый участник должен сделать 100 отжиманий в день.**  
Бот напоминает утром о вчерашних должниках (с мотивационной фразой), предупреждает вечером тех, кто ещё не выполнил норму, и ведёт статистику.

## ✨ Возможности

- 📅 **Персональная норма** – каждый участник выполняет 100 отжиманий за день.
- ⏰ **Утреннее напоминание** (по умолчанию в 8:00) – список участников, не сделавших норму вчера, + случайная мотивационная фраза.
- ⏰ **Вечернее предупреждение** (по умолчанию в 22:00) – список тех, кто сегодня ещё не набрал 100 (осталось 2 часа до полуночи).
- 📊 **Команды на русском языке** (также поддерживаются английские):
  - `/отжимания <число>` – добавить отжимания за сегодня
  - `/мои` – сколько вы сделали сегодня
  - `/статистика` – прогресс всех участников за сегодня
  - `/топ` – лидеры дня по количеству отжиманий
  - `/помощь` – справка
- 🗄️ **Хранение данных** – SQLite (результаты за каждый день, список пользователей).
- 🧠 **Расширенная база мотивации** – 46+ вдохновляющих фраз, выбирается случайно каждое утро.
- 🐍 **Совместимость** – Python 3.10 – 3.14, `python-telegram-bot` версии 22.x.

## 🚀 Запуск бота

### 1. Подготовка окружения

- **Python 3.10 – 3.14** (рекомендуется 3.11 или 3.12)
- Telegram-бот, созданный через [@BotFather](https://t.me/BotFather) (получить **токен**)
- Группа в Telegram, куда добавлен бот. **Бот должен быть администратором** (чтобы видеть всех участников).

### 2. Установка зависимостей

```bash
pip install "python-telegram-bot[job-queue]>=22.0" pytz

### 3. Настройка (подробно)
3.1. Получение токена бота
Напишите @BotFather команду /newbot

Придумайте имя бота и username (должен заканчиваться на bot)

Скопируйте полученный токен (например, 1234567890:ABCdefGHIjklmNOPqrstUVwxyz)

3.2. Определение ID группы
Существует несколько способов. Самый простой:

Добавьте в файл bot.py (временно) следующий обработчик:

python
async def chatid(update, context):
    await update.message.reply_text(str(update.effective_chat.id))
application.add_handler(CommandHandler("chatid", chatid))
Запустите бота, отправьте в группе команду /chatid

Бот ответит числом, например -1001234567890. Это и есть CHAT_ID.

3.3. Редактирование config.py
Откройте файл config.py и укажите:

python
TOKEN = "ваш_токен_от_BotFather"
CHAT_ID = -1001234567890   # число, полученное на предыдущем шаге (обязательно отрицательное)

3.4. Права администратора для бота
Зайдите в настройки группы → «Управление группой» → «Администраторы»

Добавьте бота в список администраторов. Достаточно прав:

✅ Чтение сообщений

❌ Остальные можно не включать (бот не удаляет участников)

### 4. Запуск (подробно)
4.1. Локальный запуск (для тестирования)
bash
export TELEGRAM_BOT_TOKEN="ваш_токен"
python bot.py
(на Windows: set TELEGRAM_BOT_TOKEN=ваш_токен)

4.2. Постоянная работа на сервере
Через screen:

bash
screen -S pushups_bot
export TELEGRAM_BOT_TOKEN="ваш_токен"
python bot.py
# Отключиться: Ctrl+A, затем D
Через systemd (Linux) – создайте файл /etc/systemd/system/pushups-bot.service:

ini
[Unit]
Description=Telegram Pushups Bot
After=network.target

[Service]
User=your_user
WorkingDirectory=/home/your_user/pushups_bot
Environment="TELEGRAM_BOT_TOKEN=ваш_токен"
ExecStart=/usr/bin/python3 /home/your_user/pushups_bot/bot.py
Restart=always

[Install]
WantedBy=multi-user.target
Затем выполните:

bash
sudo systemctl enable pushups-bot.service
sudo systemctl start pushups-bot.service
