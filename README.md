# pushups_bot
код Telegram-бота для контроля отжиманий в групповом чате. Бот:  каждое утро напоминает участникам о необходимости отжиматься;  принимает от пользователей количество выполненных отжиманий (команда /pushups &lt;число>);  суммирует результаты за день;  в конце дня, если общая сумма меньше 100, отправляет в чат предупреждение.
